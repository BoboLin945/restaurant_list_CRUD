## :bulb: Getting Started

#### Clone repository
```
$ git clone https://github.com/BoboLin945/restaurant_list_CRUD.git
```
#### Install by npm
```
$ npm install
```
#### DB Seed
```
$ npm run seed
```
#### Execute
```
$ npm run dev
```
#### Browse the website on localhost
```
http://localhost:3000/
```
